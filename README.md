## Installation

1. Clone this repo

  ```bash
  git clone https://github.com/Team-One/Kiosk-Proto
  ```

2. Installation

  ```bash
  npm install
  ```

3. Run

  ```bash
  npm run start
  ```

4. Build

  ```bash
  npm run build
  ```

5. Deploy

  ```bash
  npm run deploy
  ```
